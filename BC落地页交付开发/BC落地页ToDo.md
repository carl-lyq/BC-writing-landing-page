## 路径02 · AI 学习方案（已定摘要 · 详见 PRD §2.2 功能 5.1）

- **入口**：路径 02 主 CTA（落地页内嵌弹窗，**不走** T-002 外链）
- **上传**：登录后 **1 个** PDF，单文件 ≤5MB，**任意 PDF 均可**（内容不限）
- **输出**：扣子工作流 → **Markdown**；前端渲染预览 + 后端导出 PDF（见 `路径02-方案Markdown与PDF开发说明.md`）
- **生成**：异步任务，目标 **≤1 分钟**；等待中可关弹窗；完成后**当前会话内自动弹出**结果
- **回访**：再次进入页面**不自动弹**；路径 02 区展示「查看我的方案」
- **覆盖**：再次生成前确认；生成中点击 CTA → 提示「生成中」；服务端保留最近 **3 份**历史
- **失败**：扣子 / PDF / 任意环节报错 → 页顶 Toast「方案生成失败，请重试或联系客服」**3 秒**；覆盖生成失败时恢复原方案

### T-004 预设展示量（详见 PRD §2.2 / §4.2）

- **展示播放量** = 真实播放量 + round(视频预设值 × 随机系数 × 时间系数)
- **视频预设值**（6 集）：1000 / 900 / 800 / 700 / 600 / 500
- **随机系数**：∈ [0.9, 1.1]，每集独立，页面加载时生成
- **时间系数**：2026-06-01 前 = 1；自 2026-06-01 起每天 +0.05（6/1 当天仍为 1）

### T-005 真实播放量（详见 PRD §2.2 / §4.2）

- **触发**：点击开始播放计 1 次（不要求完播）
- **去重**：同一用户 + 同一视频 **仅计 1 次**，重复播放不计
- **用户标识**：已登录→账号 ID；未登录→优先 `ut`，否则持久化匿名访客 ID
- **更新频率**：**每小时**聚合/刷新
