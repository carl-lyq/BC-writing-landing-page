## 1. 设计目标（30 秒读懂）

单页长滚动落地页，承接 BC WriteUp 流量，完成：

**建立信任**（标语 + 公开课）→ **识别需求**（选卡 / 问卷）→ **展示方案**（6 路径）→ **转化**（路径 CTA / 微信 / 登录）

视觉调性：**Editorial / Warm** — 纸感背景、墨水字、蓝色 accent、编辑感排版。  
受众：中文雅思考生，**全页中文 UI**（无英文装饰标签）。

## 4. 设计 Token（与 Demo 一致）

| Token | 值 | 用途 |
|-------|-----|------|
| `--paper` | `#f4f7fb` | 页面背景 |
| `--paper-2` | `#e7eef7` | 次级背景 / 推荐路径高亮 |
| `--ink` | `#0b1f3a` | 主文字 |
| `--ink-2` | `#1e3a63` | 次要正文 |
| `--ink-3` | `#4f6a8f` | Meta、辅助标签 |
| `--accent` | `#1e5fd6` | 强调色、链接、进度 |
| `--accent-ink` | `#143f99` | 深色强调、数据数字 |
| `--accent-wash` | `#d8e5fb` | 浅蓝底纹、箭头圆底 |
| `--radius` | `16px` | 卡片、按钮圆角 |
| `--page-max` | `1440px` | 内容最大宽度 |
| `--gutter` | `clamp(16px, 2vw, 32px)` | 左右边距 |

**字体栈**

| 用途 | 字体 |
|------|------|
| 大标题 / 标语 | **Noto Serif SC**（serif） |
| 正文 / 按钮 | **Noto Sans SC**（sans） |
| 标签 / 序号 / Meta | **JetBrains Mono**（mono） |

**字号参考（已定）**

| 元素 | 字号 |
|------|------|
| 区块一级标题（问题匹配、学习路径…） | 32–48px serif，**无**「01/02」页面序号 |
| 区块二级 slogan | 24–36px serif |
| 痛点卡用户原话 | 22px serif |
| 选中路线名 | 20–24px serif 加粗 |
| Meta 小标签（入学分、tag 等） | **13px**（勿小于此） |
| Meta 中标签（讲师行等） | 14px |

## 5. 分区块设计要点

### 5.1 导航（Sticky）

| 项 | 说明 |
|----|------|
| 锚点（4 个） | 公开课 · 我的提分需求 · 学习路径 · 学员故事 |
| 右侧 CTA | 「登录/注册」；已登录显示「已登录」 |
| 联名 Logo | **BC备考平台 + 躺着学** 并排 — **Demo 暂仅躺着学，需你补全（T-008）** |
| 页脚 | **不重复** BC Logo，对齐 [躺着学主站](https://igopx.cn/index) Footer |

### 5.2 Hero · 首屏标语（v0.19 新增）

- 整宽卡片：左 **5px accent 竖条** + 浅蓝渐变底  
- **左栏** 两行 serif 标语，「一/六」与「考生/你」accent 高亮 + 浅色底纹呼应  
- **右栏** 2×2 数据标签；第 4 张为 **accent 渐变底**（7 天 SVIP）  
- 每张标签 **右侧** 低透明度抽象图标（视频 / 曲线 / 趋势 / 星形），不挡字  
- Mobile：**标语在上 → 数据标签 → 视频区**

**标语文案（已定，勿改字）**

- 一站式雅思备考平台，已助力30万**考生**出分。  
- 六类个性化学习路径，总有一种满足**你**需求。

**数据标签（已定）**

| 主数字 | 说明 |
|--------|------|
| 207个 | 雅思考点视频讲解 |
| 100万+ | 学生作文大数据 |
| 30天 | 平均提高 **1** 分 |
| 7天 | 免费 **SVIP** 全功能体验 |

### 5.3 Hero · 视频区

- 主播放器 **严格 16:9**，禁止固定高度拉伸变形  
- 右侧列表：「写作免费公开课 · 6 集」；眉标「播放列表」  
- 占位标签「精选公开课」；讲师统一 **Icey Zhang · 官方认证教师**  
- **无需登录**即可观看；**不做**自动连播开关  
- 6 集标题/小标题见 `TODO.md` T-003a；封面/时长待 **T-003b**

### 5.4 问题匹配

**区块标题**

- 一级：**问题匹配**  
- 二级 slogan：**大数据匹配最佳学习路径**

**模式切换（正式版必须有，不可放进调试面板）**

| 按钮 | 状态 |
|------|------|
| **自主选择** | 默认选中，实心 accent |
| **互动诊断** | 非选中也须显眼；带标签 **「≈1 分钟」** |
| 引导文案 | 仅在模式 A 显示，与 Tab 同行：「不确定选哪个？试试 **互动诊断** — 4 道题…」 |

**模式 A · 6 张痛点卡**

- 3×2 网格（Mobile 可 1–2 列）  
- **选中态**：深底 ink + 浅字 + **圆形勾 ✓** + 路线名放大（20–24px）+ 约 0.4s 入场动效  
- 卡片底 **仅路线名**，无「匹配 →」前缀  
- 下方匹配结果条：未选「等待选择」/ 已选「已匹配」+「查看路径」按钮  

**模式 B · 问卷**

- 左：4 题 + 顶部分段进度条（已答 accent / 当前 ink / 未答灰）  
- 右：**写作诊断表**实时预览  
- Q4 多选；完成页「查看专属路径」+「重新测试」  
- Tablet/Mobile：问卷在上，诊断表在下  

### 5.5 学习路径（6 条）

**区块 slogan**：一条路线，对应一种学习状态。

| 序号 | kicker | 主标题（serif） |
|------|--------|-----------------|
| 01 | 课程学习路径 | 把好老师的课堂,搬到你的书桌上。 |
| 02 | 专属提分方案 | 上传写作报告，AI 生成你的学习方案 |
| 03 | 自学 + 互动课 | 不是在看视频, 是在课堂里。 |
| 04 | AI 学习计划 | 把目标分数, 拆成每天做什么。 |
| 05 | 学习营/导师 | 一个人写容易停, 有人陪就不一样。 |
| 06 | 人工精批/诊断 | 人工精批诊断，让你真正看懂批改报告 |

- 顶部 **6 列索引条**；匹配完成后对应列 + 详情区 **paper-2 背景高亮**（**无**「推荐给你」角标）  
- 详情区：媒体 / 文案 **左右交替**；右上角 **「回到路径选择」** ghost 按钮  
- 每条：主 CTA 按钮 + **微信咨询卡**  
  - 眉标 mono：「微信 · 扫码咨询」  
  - 主文案：「详情添加老师咨询」  
  - **不要**放微信号、服务时间  
- 路径 01 bullets 含「7 天免费试学」— 是路径卖点，**不是**独立试学区块  

**路径 02 弹窗（需出 5 态稿）**

| 态 | 标题 | 要点 |
|----|------|------|
| 上传 | 上传写作报告 | 1–2 个 PDF 槽位，≤5MB 提示 |
| 生成中 | 正在生成方案 | 进度条 +「可关闭，后台继续」 |
| 覆盖确认 | 重新生成方案？ | 取消 / 确认覆盖 |
| 结果 | 你的 AI 学习方案 | 预览区 + 下载 PDF |
| 回访 | — | 路径区 secondary「查看我的方案 →」 |

路径 02 详情文案为**草案**（T-014），可先按 Demo，后续可能微调。

### 5.6 登录 / SVIP 弹窗

- **登录/注册弹窗**：accent 底横幅 —「注册即送价值￥99的7天 **SVIP** 会员…」；「SVIP」可点击  
- 由路径 CTA 触发时加一行：「登录后继续：{CTA 文案}」  
- **SVIP 介绍弹窗**：复用主站风格；Demo 为占位列表  
- 正式版 **复用主站**登录组件视觉，本页仅定布局与文案区层级  

### 5.7 学员故事

- 一级：**学员故事** · slogan：**真实反馈，不加修饰**  
- 3 列卡（Mobile 单列）：**入学分**（删除线）→ 圆形 accent 箭头 → **实考出分**（大字号）  
- 周期：**学习周期：N周**（勿用 BEFORE/AFTER 等英文）  
- 三例数据见 PRD §9.7；营销表述 **已确认可用**

### 5.8 页脚

- **全宽对齐主站** [`igopx.cn`](https://igopx.cn/index)：深色背景图 + 左栏公司信息 + 图中含官方账号与联系 QR  
- 左栏文案：雅思就要 躺着学 / 主办单位 / 地址 / ICP（链至 beian.miit.gov.cn）  
- 背景图资源 URL 见 Demo `.site-footer__bar`（正式走 CDN）

### 5.9 左侧流程轴（Desktop ≥1280px）

- 4 主步 + 「学习路径」下 6 条子路径（01–06 kicker）  
- **无**总标题「页面流程」、**无**各步副标题  
- 1280–1819px：主内容左预留槽位防遮挡；≥1820px 轴在页边距区  
- **&lt;1280px 整块隐藏**，仅保留顶部导航

## 6. 响应式必做清单

设计稿须至少覆盖 **三档**：

| 档位 | 宽度 | 你需重点核对 |
|------|------|--------------|
| Mobile | &lt; 768px | Hero 上下堆叠；痛点卡列数；路径索引横滑；弹窗全宽 |
| Tablet | 768–1023px | Hero 双栏或堆叠；问卷/诊断表堆叠 |
| Desktop | ≥ 1024px | 3×2 痛点；6 列路径索引；3 列学员故事 |
| 宽屏 | ≥ 1280px | 流程轴 + 主内容左间距 |
