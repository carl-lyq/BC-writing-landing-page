## 1. 设计目标（30 秒读懂）

单页长滚动落地页，承接 BC WriteUp 流量，完成：

**建立信任**（标语 + 公开课）→ **识别需求**（选卡 / 问卷）→ **展示方案**（6 路径）→ **转化**（路径 CTA / 微信 / 登录）

视觉调性：**Cool Paper / Route Map** — 冷色纸面、Ink 字、Route Blue accent（见正式 PRD §2.5）。  
受众：中文雅思考生，**全页中文 UI**（无英文装饰标签）。

## 3. 页面结构（4 区块 · 无底部试学区）

```
┌─ Sticky 导航 ─────────────────────────────────────┐
│ [躺着学 Logo]  4锚点  ·········  [登录/注册] [BC Logo] │
├─ #hero 公开课 ────────────────────────────────────┤
│  首屏标语（左文案 + 右 2×2 数据标签）              │
│  主播放器 16:9 + 右侧 6 集播放列表                 │
├─ #pain 问题匹配 ──────────────────────────────────┤
│  Tab：自主选择 | 互动诊断（≈1 分钟）              │
│  模式 A：6 痛点卡 3×2  ·  模式 B：问卷 + 诊断表   │
├─ #paths 学习路径 ─────────────────────────────────┤
│  6 列索引条 → 6 条路径详情（媒体/文案交替）        │
│  含登录弹窗、路径 02 AI 方案弹窗（5 态）           │
├─ #proof 学员故事 ─────────────────────────────────┤
│  3 列案例卡                                        │
├─ 页脚（对齐躺着学主站）───────────────────────────┤
└───────────────────────────────────────────────────┘

Desktop ≥1280px：左侧另有固定「流程轴」（4 主步 + 6 路径子项）
```

**重要：已移除** 原页面底部「免费试学」深色转化区（`#cta`）。  
转化收口在：**6 条路径主 CTA**、**路径内微信咨询**、**导航登录/注册**。

**入口**：本页在 **主站隐藏**，主站不得链入；用户仅经 **WriteUp 深链** 或直达 URL 进入。

## 4. 设计 Token（与 Demo 一致）

| Token | 值 | 用途 |
|-------|-----|------|
| `--paper` | `#f4f7fb` | 页面背景 |
| `--paper-2` | `#e7eef7` | 次级背景 / 推荐路径高亮 |
| `--ink` | `#0b1f3a` | 主文字 |
| `--ink-2` | `#1e3a63` | 次要正文 |
| `--ink-3` | `#4f6a8f` | Meta、辅助标签 |
| `--accent` | `#1e5fd6` | 强调色、链接、进度 |
| `--accent-ink` | `#143f99` | 深色强调、数据数字 |
| `--accent-wash` | `#d8e5fb` | 浅蓝底纹、箭头圆底 |
| `--radius` | `16px` | 卡片、按钮圆角 |
| `--page-max` | `1440px` | 内容最大宽度 |
| `--gutter` | `clamp(16px, 2vw, 32px)` | 左右边距 |

**字体栈**

| 用途 | 字体 |
|------|------|
| 大标题 / 标语 | **Noto Serif SC**（serif） |
| 正文 / 按钮 | **Noto Sans SC**（sans） |
| 标签 / 序号 / Meta | **JetBrains Mono**（mono） |

**字号参考（已定）**

| 元素 | 字号 |
|------|------|
| 区块一级标题（问题匹配、学习路径…） | 32–48px serif，**无**「01/02」页面序号 |

| 痛点卡用户原话 | 22px serif |

## 5. 分区块设计要点

### 5.1 导航（Sticky）

| 项 | 说明 |
|----|------|
| 锚点（4 个） | 公开课 · 我的提分需求 · 学习路径 · 学员故事 |
| 右侧 CTA | 「登录/注册」；已登录显示「已登录」 |
| 品牌区 | **左**躺着学 Logo 89×34（Mobile 73×28）· **右**登录/注册 + **BC备考平台** Logo（**高度与躺着学一致** 34/28，宽等比；登录在 BC 左侧） |
| 页脚 | **不重复** BC Logo，对齐 [躺着学主站](https://igopx.cn/index) Footer |

### 5.2 Hero · 首屏标语（v0.19 新增）

- 整宽卡片：左 **5px accent 竖条** + 浅蓝渐变底  
- **左栏** 两行 serif 标语，「一/六」与「考生/你」accent 高亮 + 浅色底纹呼应  
- **右栏** 2×2 数据标签；第 4 张为 **accent 渐变底**（7 天 SVIP）  
- 每张标签 **右侧** 低透明度抽象图标（视频 / 曲线 / 趋势 / 星形），不挡字  
- Mobile：**标语在上 → 数据标签 → 视频区**

**标语文案（已定，勿改字）**

- 一站式雅思备考平台，已助力30万**考生**出分。  
- 六类个性化学习路径，总有一种满足**你**需求。

**数据标签（已定）**

| 主数字 | 说明 |
|--------|------|
| 207个 | 雅思考点视频讲解 |
| 100万+ | 学生作文大数据 |
| 30天 | 平均提高 **1** 分 |
| 7天 | 免费 **SVIP** 全功能体验 |

### 5.3 Hero · 视频区

- 主播放器 **严格 16:9**，禁止固定高度拉伸变形  
- 右侧列表标题：**写作免费公开课 · 6 集**（**无**独立眉标「播放列表」）+ 右侧总时长  
- 占位标签「精选公开课」；讲师统一 **Icey Zhang · 官方认证教师**  
- **无需登录**即可观看；**不做**自动连播开关  
- 6 集标题/小标题见 `TODO.md` T-003a；封面/时长/MP4 已接入 **T-003b**（`UI/assets/hero/`）

### 5.4 问题匹配

**区块标题**（详见 PRD §2.2 功能 2–4）

- **无**页面内一级标签「问题匹配」（导航锚点仍为「我的提分需求」）  
- **主标题（两行、居中）**：最短路径匹配 / 避免碎片化无效学习  
- **辅助说明**（略大字号）：基于 **30万** / **20年** / **6条**（accent 强调）…写作提分路径，适配所有学生需求

**模式切换（正式版必须有，不可放进调试面板）**

| 按钮 | 状态 |
|------|------|
| **自主选择** | 默认选中，实心 accent |
| **互动诊断** | 非选中也须显眼；带标签 **「≈1 分钟」** |
| 引导文案 | 仅在模式 A 显示，与 Tab 同行：「不确定选哪个？试试 **互动诊断** — 4 道题…」 |

**模式 A · 6 张痛点卡**

- 3×2 网格（Mobile 可 1–2 列）  
- **选中态**：Route Blue `#1E5FD6` 底 + 白字；无右上角箭头；无底部「已匹配」横条  
- 卡片底 **仅路线名**；**选中卡右下角**「查看路径」胶囊（平滑滚动至 `#path-{hookIdx}`）  

**模式 B · 问卷**

- 左：4 题 + 顶部分段进度条（已答 accent / 当前 ink / 未答灰）  
- 右：**写作诊断表**实时预览  
- Q4 多选；完成页「查看专属路径」+「重新测试」  
- Tablet/Mobile：问卷在上，诊断表在下
