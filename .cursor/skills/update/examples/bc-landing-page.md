# 示例：BC WriteUp 雅思写作落地页

> 本文件为 **BC落地页** 项目的主源映射与水滴分布清单；其他项目可复制 [reference.md](../reference.md) 自建 `examples/<project>.md`。

## 主源判定（本项目）

| 角色 | 路径 | 说明 |
|------|------|------|
| **主 PRD（工作稿）** | `UI/docs/BC落地页PRD_UI更新.md` | 产品持续更新 v1.0.x |
| **开发冻结 PRD** | `BC落地页PRD_6.5.md` | 仅用户要求「同步冻结稿」时更新 |
| **归档 PRD** | `BC落地页PRD.md` | v0.21，**不更新** |
| **主 HTML** | `UI/index.html` | UI/交互/水滴基准 |
| **交付镜像** | `BC落地页交付开发/UI/index.html` | 工作区改完后 `cp` 同步 |
| **TODO** | `BC落地页ToDo.md` | 与 `PAGE_TODOS` 对齐 |
| **历史 Demo** | `BC落地页Demo.html` | **非验收基准** |

**开发交付包**（`BC落地页交付开发/`）给开发用：只含 `BC落地页PRD_6.5.md`，**不含**工作稿 PRD（避免多版本误解）。

## PAGE_TODOS 规则

1. 键名 = `BC落地页ToDo.md` 表格 ID（如 `T-003b`）
2. `prd` 数组须在主 PRD 中存在
3. 已定稿：更新 `status` 或移除 `TodoDrip`

## /update 后可选同步

| 动作 | 条件 |
|------|------|
| `cp UI/index.html → BC落地页交付开发/UI/` | 默认 |
| 更新 `BC落地页PRD_6.5.md` | 用户要求或开发定稿变更 |
| 更新 `BC落地页交付开发/BC落地页ToDo.md` | 与根 TODO 保持一致时 |
